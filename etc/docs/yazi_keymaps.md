# Yazi Keymaps

## Navigation

| Key | Description |
|-----|-------------|
| `l` | Enter directory (auto-skips single-child dirs) or open file |
| `1`-`9` | Relative motion (jump N lines) |
| `F` | Smart filter |
| `z` | Jump to directory with zoxide |
| `Ctrl-f` | Fuzzy find file with fzf |

## Quick Navigation (g + key)

| Key | Description |
|-----|-------------|
| `g h` | Home directory |
| `g d` | ~/Downloads |
| `g D` | ~/Desktop |
| `g c` | ~/.config |
| `g .` | Dotfiles |
| `g p` | ~/Programming/JimmyTranDev |
| `g b` | ~/Programming/storebrand-digital |
| `g w` | ~/Programming/wcreated (worktrees) |

## Bookmarks

| Key | Description |
|-----|-------------|
| `m` | Save bookmark |
| `'` | Jump to bookmark |
| `b d` | Delete bookmark |
| `b D` | Delete all bookmarks |

## File Operations

| Key | Description |
|-----|-------------|
| `Alt-y` | Copy file contents to clipboard |
| `c m` | Change file permissions (chmod) |
| `Alt-d` | Diff selected file with hovered file |

## View

| Key | Description |
|-----|-------------|
| `T` | Toggle maximize preview pane |
